# Sistema de Automação por Reconhecimento de Voz

Este projeto integra Python e Arduino para controle residencial via comandos de voz utilizando processamento em nuvem.

---
## Estrutura do Projeto

- server/: Lógica em Python e processamento de voz.
- hardware/: Código .ino para o Arduino.
- requirements.txt: Bibliotecas necessárias.

---
## Como Configurar

1. Instale as dependências com: pip install -r requirements.txt
2. Carregue o código na pasta hardware para seu Arduino.
3. Execute o arquivo server/main.py.

---
## Comandos de Voz

- Ligar luz
- Desligar luz
- Ligar sirene
